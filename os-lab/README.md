# OS Lab App

This folder contains the Next.js app for the **OS + AOA Interactive Learning Platform**.

For the full repository overview, setup notes, and project explanation, see:

- [../README.md](../README.md)

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev`
- `npm run build`
- `npm start`
- `npm run lint`

## Main Routes

- `/` - landing page
- `/combined` - integrated OS + AOA simulation
- `/os` - disk scheduling mode
- `/aoa` - algorithm analysis mode
